ABOUT - BLOG PESSOAL DEVELOPMENT

Here you can acknowledge the development of Blog Pessoal!

Blog Pessoal was developed with Java and Spring Boot and idealized as an application where it would be possible to write posts and see other's posts as a goal to share thoughts and worries, like a safe place where you can feel confortable and know that you are not alone. Although, to avoid the oposite result, people can not interact directly with posts. The final project was released after Front-end was fully complete using Angular Framework giving Blog Pessoal this friendly interface that you are able to navigate to! Hope you have enjoyed using it, do not forget to read more about the developer and the application at the footer below and menu above!
